A DepEd-aligned assessment report generator with subscription management system.

## Features

### Main Assessment Tool (index.html)
- 📋 Assessment information input
- 👨‍🎓 Learner performance data entry
- 📝 Item-by-item analysis
- 📎 Competency mapping
- 📊 Automatic report generation with charts
- 📈 MPS, mastery levels, and item difficulty analysis
- 🖨️ Print/PDF export capability

### Admin Management Panel (admin.html)
- 🔐 Secure login (password protected)
- 🔑 Subscription code generation
- 📊 Dashboard with statistics
- 📤 Export codes to CSV
- 🗑️ Code management (add/delete)

## How to Use

1. **For Users:**
   - Open `index.html` in a browser
   - Pay subscription fee via GCash: **0935 836 0471**
   - Enter the subscription code provided by the administrator
   - Generate assessment reports

2. **For Administrators:**
   - Open `admin.html` in a browser
   - Login with administrator password
   - Generate subscription codes for users
   - Manage and track subscriptions

## Technology Stack
- HTML5, CSS3, JavaScript (Vanilla)
- Chart.js for data visualization
- localStorage for data persistence
- Responsive design for all devices
